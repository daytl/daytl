# Daytl

[Daytl.com](https://www.daytl.com) is an open source project built on [Gatsby.js]().

## Tools
* Czech BirthNumber tool
* Remover
* Base64 decoder/emcoder inc. image support

### Dev in progress

* JWT parser, case convertor


### Upcoming tools

* Encode/decode - utf, url encode, html
* formatter
* minification
