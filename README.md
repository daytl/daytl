# Daytl

[Daytl.com](https://www.nextjs.org) is an open source project built on [Next.js]().

## Tools
* Czech BirthNumber tool
* Remover
* Base64 decoder/encoder inc. image support
* JWT parser

### Dev in progress

* QR code generator

### Upcoming tools
* Encode/decode - utf, url encode, html
* Formatter
* Minification
* Case convertor
